# MQTT Skill

The `mqtt` skill resolves values from an MQTT broker topic.

## Config
- `name`
- `brokerHost`
- `brokerPort`
- `topic`
- `jsonKey`
- `timeout`

## Generic Delivery
- `sendMode`
- `displayDuration`
- `template`

## Output
The MQTT skill resolves the latest available topic payload or extracted JSON value as a normalized output string.
